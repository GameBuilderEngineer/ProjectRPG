■キャラクター仕様
　■WASD / 左スティック				：移動
　□Shiftキー(長) / □ボタン(長)	：ダッシュ
　□Shiftキー(短) / □ボタン(短)	：瞬間移動ステップ
　■Spaceキー / ×ボタン			：ジャンプ
　□左クリック / R1ボタン			：攻撃（銃/剣）
　□Qキー / L1ボタン				：武器切り替え
　□マウス座標 / 右スティック		：カメラ
　□右クリック / R2ボタン			：スキル（ガラクタ引き寄せ）
　□Ctrl / 右スティック押し込み		：ロックオン
